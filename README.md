Compressor
==========
Client-side offline-first JavaScript/CSS/HTML minifier. Just load it once and it will available for offline use.

Visit: https://compressor.whe.technology


## Installation
```bash
$ npm install && npm run build
```

Start development server:
```bash
$ npm run watch
```
